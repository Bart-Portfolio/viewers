360° Rotation Viewer

Frame-sequence turntable

A classic ecommerce-style 360° spin, built from a sequence of still photographs rather than a 3D model. Drag across the image, or use the slider below, to rotate through every angle — the same interaction shoppers expect from product pages like this one.

Format
PNG frame sequence
Frame count
20 images, 18° apart
Interaction
Drag or slider
Source folder
assets/360/

Drop your own sequence into assets/360/ as frame_001.png … frame_020.png — see the README there for setup notes.

glTF Model Viewer

Real-time Three.js render

A real-time 3D viewer built with Three.js, loading an actual glTF/GLB model rather than pre-rendered photos. Orbit around the product freely and zoom in — useful for showing off geometry and material detail a flat photo sequence can't capture.

Format
glTF / GLB (binary)
Engine
Three.js, real-time render
Interaction
Orbit drag, scroll to zoom
Source file
assets/models/product.glb

Replace assets/models/product.glb with your own model — any reasonably optimized glTF/GLB will load automatically.
